前言

airflow是一个以编程的方式编写、调度和监控工作流的平台，并支持与第三方平台的集成。

  • 2014年，诞生于Airbnb
  • 2016年，进入Apache孵化器
  • 2019年，1月8日毕业成为Apache顶级项目

安装

安装选用的是airflow最新的版本，1.10.3。

安装过程中如果有依赖包的版本较新，无法通过pip源安装，可以通过github安装。
