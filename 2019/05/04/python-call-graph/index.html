python call graph

Posted on 2019-05-04

介绍

pycallgraph被设计成一个用于python应用程序的可视化分析工具。它使用一个名为sys.set_trace()的调试python函数，每次代码进入或离开函数时都会进行回调。这允许pycallgraph跟踪每个被调用函数的名称，以及调用哪个函数、每个函数内所用的时间、调用次数等。
